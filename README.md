# Flutter To-Do App

A simple and beginner-friendly **Flutter To-Do List app** to manage daily tasks.

## Features
- Add new tasks
- Mark tasks as done/undone using checkboxes
- Delete tasks
- Simple and clean UI
- Built with Flutter (Dart)
